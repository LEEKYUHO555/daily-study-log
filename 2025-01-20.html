JavaScript ES6+ 기초

JavaScript ES6 Frontend

Arrow Functions, Destructuring, Template Literals 등 ES6의 주요 기능들을 학습했습니다. 모던 JavaScript 개발에 필수적인 문법들을 실습을 통해 익혔습니다.

📚 학습 내용

오늘은 ES6(ECMAScript 2015)에서 도입된 주요 기능들을 학습했습니다. ES6는 JavaScript의 큰 변화를 가져온 버전으로, 현대적인 JavaScript 개발의 기초가 되는 문법들을 제공합니다.

1. Arrow Functions (화살표 함수)

화살표 함수는 함수를 더 간결하게 작성할 수 있게 해주며, this 바인딩이 기존 함수와 다르게 동작합니다.

2. Destructuring (구조 분해 할당)

배열이나 객체의 속성을 해체하여 그 값을 개별 변수에 담을 수 있게 해주는 JavaScript 표현식입니다.

3. Template Literals (템플릿 리터럴)

백틱(`)을 사용하여 문자열을 생성하며, 문자열 보간과 여러 줄 문자열을 쉽게 만들 수 있습니다.

4. Let과 Const

블록 스코프를 가지는 새로운 변수 선언 방식으로, var의 문제점들을 해결합니다.

💻 코드 예제

Arrow Functions

// 기존 함수 표현식
const add = function(a, b) {
    return a + b;
};

// 화살표 함수
const add = (a, b) => a + b;

// 여러 줄 화살표 함수
const multiply = (a, b) => {
    const result = a * b;
    return result;
};

// 배열 메서드와 함께 사용
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(n => n * 2);
console.log(doubled); // [2, 4, 6, 8, 10]

Destructuring

// 배열 구조 분해
const [first, second, ...rest] = [1, 2, 3, 4, 5];
console.log(first); // 1
console.log(second); // 2
console.log(rest); // [3, 4, 5]

// 객체 구조 분해
const person = {
    name: 'John',
    age: 30,
    city: 'Seoul'
};

const { name, age, city } = person;
console.log(name); // 'John'

// 기본값 설정
const { name, age, country = 'Korea' } = person;
console.log(country); // 'Korea'

// 함수 매개변수에서 구조 분해
function greet({ name, age }) {
    console.log(`안녕하세요, ${name}님! ${age}세이시군요.`);
}

greet(person); // 안녕하세요, John님! 30세이시군요.

Template Literals

const name = 'JavaScript';
const version = 'ES6';

// 문자열 보간
const message = `안녕하세요, ${name} ${version}입니다!`;
console.log(message); // 안녕하세요, JavaScript ES6입니다!

// 여러 줄 문자열
const multiLine = `
    첫 번째 줄
    두 번째 줄
    세 번째 줄
`;

// 표현식 사용
const a = 5;
const b = 10;
console.log(`${a} + ${b} = ${a + b}`); // 5 + 10 = 15

// 함수 호출
function formatCurrency(amount) {
    return `₩${amount.toLocaleString()}`;
}

const price = 1000000;
console.log(`가격: ${formatCurrency(price)}`); // 가격: ₩1,000,000

Let과 Const

// var의 문제점
function varExample() {
    if (true) {
        var x = 1;
    }
    console.log(x); // 1 (함수 스코프)
}

// let과 const의 블록 스코프
function letConstExample() {
    if (true) {
        let y = 1;
        const z = 2;
    }
    // console.log(y); // ReferenceError
    // console.log(z); // ReferenceError
}

// const는 재할당 불가
const PI = 3.14159;
// PI = 3.14; // TypeError

// 하지만 객체의 속성은 변경 가능
const person = { name: 'John' };
person.name = 'Jane'; // 가능
person.age = 25; // 가능

🔑 핵심 포인트

  • Arrow Functions: 간결한 문법과 lexical this 바인딩이 특징
  • Destructuring: 코드를 더 읽기 쉽고 간결하게 만들어줌
  • Template Literals: 문자열 조작이 훨씬 편리해짐
  • Let/Const: 블록 스코프로 더 예측 가능한 코드 작성 가능
  • ES6+ 문법은 현대 JavaScript 개발의 표준
  • Babel 등의 트랜스파일러로 구형 브라우저 지원 가능
